# CentralContext Agent Bootstrap Prompt

Trước khi bắt đầu bất kỳ hoạt động phân tích, trả lời câu hỏi, lập kế hoạch hoặc thực thi mã nguồn nào liên quan đến các dự án trong hệ sinh thái devflow:

1. Bạn **bắt buộc** phải đọc và hiểu toàn bộ nội dung của gói **CentralContext Pack** được cung cấp bởi nhà phát triển.
2. Nếu gói ngữ cảnh này chưa được cung cấp trong lược sử chat hiện tại, bạn **phải dừng lại** và yêu cầu người dùng cung cấp hoặc hướng dẫn người dùng chạy lệnh sau trong Terminal để lấy ngữ cảnh mới nhất:
   ```bash
   npm run context:pack
   ```
3. **Tuyệt đối không được tự ý dự đoán**, giả định hoặc sử dụng kiến thức bên ngoài về các dự án: `qlythuexe`, `aimemory`, `SaveX`, `GiveGet`, `centalcontext` khi chưa có gói ngữ cảnh này.
4. Mọi quyết định kiến trúc cốt lõi phải tuân theo hướng dẫn ghi trong tài liệu `CENTRAL_CONTEXT.md` và `DECISIONS.md` đi kèm.
