# Agent Compliance Rules (Quy tắc Tuân thủ Agent)

Nếu chưa đọc CentralContext Pack:
* **Không được trả lời bất kỳ câu hỏi nào liên quan đến chiến lược** của hệ sinh thái devflow hoặc các dự án con.
* **Bắt buộc phải dừng lại và yêu cầu người dùng cung cấp** hoặc hướng dẫn chạy `npm run context:pack` / `npm run context:copy` để lấy Context Pack.
* Tuyệt đối không tự ý giả định, đoán mò hoặc đưa ra kế hoạch khi chưa có dữ liệu ngữ cảnh thực tế.
